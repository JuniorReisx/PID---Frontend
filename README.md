# PID — Intelligent Platform for Energy Transition Data

## 📌 About the Project

**PID** is an interactive web platform developed to simplify the visualization and understanding of Brazil’s energy transition. The solution combines smart maps, interactive dashboards, Artificial Intelligence, and gamification to transform technical and territorial data into accessible information for different audiences.

The platform was designed to democratize access to information about energy infrastructure, sustainability, renewable energy, and energy curtailment, offering personalized and intuitive experiences for citizens, investors, governments, and technical consultants.

---

# 🚀 Project Goal

The main goal of PID is to make complex energy transition data more visual, accessible, and easier to understand, supporting both awareness and strategic decision-making.

# 🧩 How It Works

When accessing the platform, users can select a navigation persona according to their profile and goals. Based on this selection, the system customizes the experience by displaying:

* Personalized indicators
* Smart insights
* Interactive maps
* Dynamic dashboards
* Strategic recommendations

The platform also includes an AI chatbot for answering questions and an educational game focused on sustainability and renewable energy awareness.
